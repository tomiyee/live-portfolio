How to Use the Sudoku Solver

To enter in any sudoku board to the solver, you click on one of the nine numbers below the canvas. Then, select any cells that you want to assign the number to.

If any cells are conflicting, the numbers will be highlighted in red. If any cells are red, it should be noted that the solver would be unable to find a valid solution.

Buttons

  • Begin Solution - Starts the animation where the algorithm attempts to solve the provided sudoku. See About for more details.
  • Full DFS Solution - Starts the animation where the algorithm attempts to solve the provided sudoku using only the backtrack recursion animation.
  • Quick Solve - Tries to Solve it immediately, without an animation.
  • Step - Does a single step of part 1 of the algorithm, where the algorithm assigns a cell using the hard-coded rules described in the About section.
  • Restart Board - Clears ONLY the cells edited by the algorithm, leaving the starting cells intact.
  • Clear Board - Clears ALL of the cells, including the cells provided by the user.

The Algorithm

I have a handful of hard coded algorithms to find cells which can be confidently assigned. Once none of these hard-coded algorithms can find a cell to assign, it will begin to run a depth-first search backtracking algorithm to find a possible solution.

Currently only the following two algorithms are defined before it begins the backtracking method.

Method 1 - Unique Candidates

You know that each block, row and column on a Sudoku board must contain every number between 1 and 9. Therefore, if a number, say 4, can only be put in a single cell within a block/column/row, then that number is guaranteed to fit there. This example illustrates the number 4 as the unique candidate for the cell marked in red.

Method 2 - Eight-in-a-Block/Column/Row

There is a smarter way to do this, but the simplest was to check if every other cell in a given cell's row, column or block is occupied, in which case, the remaining number in the row, column, or block must be in this cell.
