Element Finder

March 10, 2014

  • Context
  • Project

In middle school, my science teacher assigned very tedious homework assignments for chemistry. For a couple weeks, he would hand out worksheets with tables on them. Each row would have one or two cells filled in with information (e.g. 10 protons, or the Element's name), and you would be tasked with finding the element on the periodic table and completing the row.

My friend Issac and I decided that we actually wanted to automate this task of searching the periodic table, and we did so by hard coding the entire periodic table as a huge list, where every item is a list with data on the element.

I did my best to preserve the original styling of the project.
